按需安装其他依赖 1、fastclick (移动端300毫秒延迟) 1) 先执行安装fastclick的命令。 npm install fastclick -S (npm install --save fastclick) 2) 之后，在main.js中引入，并绑定到body import FastClick from 'fastclick' FastClick.attach(document.body) 2、axios (获取数据) 1) 安装依赖 npm install --save axios 2) 在main.js中引入 import axios from 'axios' Vue.prototype.$http = axios (根据个人习惯，可加可不加) 3、vux 1) npm install vux --save 2) 安装vux-loader （这个vux文档似乎没介绍，当初没安装结果报了一堆错误） 3) npm install vux-loader --save 4) 在build/webpack.base.conf.js 文件进行配置 const vuxLoader = require('vux-loader') const webpackConfig = originalConfig // 原来的 module.exports 代码赋值给变量 webpackConfig module.exports = vuxLoader.merge(webpackConfig, { plugins: ['vux-ui'] }) 4、less 1) npm install less less-loader --save 2) 修改webpack.base.conf.js文件，配置loader加载依赖，让其支持外部的less,在原来的代码上添加 { test: /\.less$/, loader: "style-loader!css-loader!less-loader" } 3) App.vue中 style标签里加上lang=”less” 4) style内引入less文件路径，例如：@import './assets/less/common.less'; 也可以直接引入less文件，在index.html引入后，需要引入less.js 5、引入font-awesome图标 1) npm install font-awesome --save引入依赖 2) main.js引入 font-awesome/css/font-awesome.min.css 即可。 例如：import './assets/font-awesome/css/font-awesome.min.css' //字体图标
